# Nexus: Android Debugger

**Debug Android C/C++ native code right inside VS Code — light, fast, and stable.**

- 🐛 **Report a bug / request a feature:** [Open an issue](https://github.com/huozhou/nexus-android-debugger-page/issues)

## What is Nexus?

Nexus brings Android **C/C++ native debugging** into VS Code, so you don't have to keep a heavy IDE open just to set a breakpoint. It connects to your device, lets you pick a process, and gives you VS Code's full debugging experience on native code.

## Features

- **Attach in a couple of clicks** — configure once, pick the process, and you're debugging on the device.
- **Light & responsive** — a lightweight, stable connection that stays out of your way; no laggy IDE, no dropped sessions.
- **Full VS Code debugging** — breakpoints, variables, call stack, watch, and the debug console on your Android native code.
- **CMake-friendly workflow** — keep using your existing CMake / incremental-build setup instead of waiting on full IDE rebuilds.
- **Localized UI in 10 languages** — English, 简体中文, 繁體中文, 日本語, 한국어, Deutsch, Français, Español, Русский, Português. The UI follows your VS Code display language automatically.

## Requirements

- A connected Android device or emulator with USB debugging enabled, running a debuggable build
- Android NDK installed
- A CMake-based native project
